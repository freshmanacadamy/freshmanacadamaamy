Question 1 of 25
1. What is the primary function of the Arithmetic and Logic Unit (ALU)?
The ALU performs calculations (like addition) and comparisons (like greater than). It's a key component of the CPU.
2. Which generation of programming languages introduced English-like syntax?
Third generation languages (like COBOL and BASIC) introduced English-like syntax, making them easier to use than assembly language.
3. What is the key difference between procedural and object-oriented programming?
OOP organizes code as objects (combining data and methods), while procedural programming organizes code as procedures/functions that operate on data.
