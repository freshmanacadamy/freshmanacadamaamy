Chapter 1: Introduction to C++

Fundamentals of C++ programming language

1.1 What is C++?

C++ is a general-purpose programming language created by Bjarne Stroustrup as an extension of the C programming language. It is widely used for system/application software, game development, drivers, and embedded systems.

Key Features:
  • Object-oriented programming support
  • High performance and efficiency
  • Rich standard library (STL)
  • Memory management control
  • Cross-platform development

1.2 Setting Up Development Environment

To start programming in C++, you'll need:

For Windows:
  1. Download MinGW compiler
  2. Install an IDE like VS Code or CLion
  3. Configure environment variables
For Linux:
sudo apt update
sudo apt install g++

1.3 Your First C++ Program

The traditional "Hello World" program in C++:

#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
Code Explanation:
Code Explanation
#include <iostream> Includes the standard input/output library
int main() Main function where execution begins
std::cout Standard output stream object
return 0; Indicates successful program termination

1.4 Basic Syntax Elements

Variables and Data Types:
int age = 25;               // Integer
double price = 99.95;       // Floating point
char grade = 'A';           // Character
bool is_valid = true;       // Boolean
std::string name = "John";  // String

Chapter Exercises

Exercise 1: Basic Output

Write a C++ program that displays your name, age, and university on separate lines.

#include <iostream>
#include <string>

int main() {
    std::string name = "Your Name";
    int age = 20;
    std::string university = "Freshman Academy";
    
    std::cout << "Name: " << name << std::endl;
    std::cout << "Age: " << age << std::endl;
    std::cout << "University: " << university << std::endl;
    
    return 0;
}
